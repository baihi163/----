## 一、核心知识点

### 1.1 三角级数与正交函数系

**三角级数（教材公式 4）：**

$$\frac{a_0}{2} + \sum_{n=1}^{\infty}\bigl(a_n\cos nx + b_n\sin nx\bigr)$$

**三角函数系（公式 5）：**

$$\{1,\;\cos x,\;\sin x,\;\cos 2x,\;\sin 2x,\;\ldots,\;\cos nx,\;\sin nx,\;\ldots\}$$

均以 $2\pi$ 为共同周期。

---

#### 正交性：从向量到函数

| 概念 | 有限维向量 | 无限维函数（$[-\pi,\pi]$） |
|------|-----------|---------------------------|
| 元素 | $\vec{u}=(u_1,u_2,\ldots)$ | $f(x)$ 在每个 $x$ 处的取值 |
| 对应相乘 | $u_i v_i$ | $f(x)\cdot g(x)$（逐点相乘） |
| 全部加起来 | $\sum u_i v_i$（点积） | $\displaystyle\int_{-\pi}^{\pi} f(x)g(x)\,dx$ |
| 正交条件 | 点积 $=0$ | 积分 $=0$ |

> **大白话：** 积分 $\int f(x)g(x)\,dx$ 就是两个「超级向量」的点积。积分为 0 → 两函数**正交**（互相垂直、不掺和）。

**几何图像：** 每个基函数像 Hilbert 空间的一根坐标轴——$1$ 是 x 轴，$\cos x$ 是 y 轴，$\sin x$ 是 z 轴……彼此垂直。

---

#### 教材正交公式（必背）

**公式 (6)：** $\displaystyle\int_{-\pi}^{\pi}\cos nx\,dx = 0$，$\displaystyle\int_{-\pi}^{\pi}\sin nx\,dx = 0$（$n=1,2,\ldots$）

**公式 (7) — 不同基函数相乘积分为 0：**

$$\int_{-\pi}^{\pi}\cos mx\cos nx\,dx = 0 \;(m\ne n), \quad \int_{-\pi}^{\pi}\sin mx\sin nx\,dx = 0 \;(m\ne n), \quad \int_{-\pi}^{\pi}\cos mx\sin nx\,dx = 0$$

**公式 (8) — 自己乘自己积分为 $\pi$（或 $2\pi$）：**

$$\int_{-\pi}^{\pi}\cos^2 nx\,dx = \pi, \quad \int_{-\pi}^{\pi}\sin^2 nx\,dx = \pi, \quad \int_{-\pi}^{\pi}1^2\,dx = 2\pi$$

**汇总表：**

| 积分 | $m \ne n$ | $m = n \ne 0$ | 常数 $1\cdot 1$ |
|------|-----------|---------------|----------------|
| $\int \cos mx\cos nx\,dx$ | $0$ | $\pi$ | — |
| $\int \sin mx\sin nx\,dx$ | $0$ | $\pi$ | — |
| $\int \cos mx\sin nx\,dx$ | $0$ | — | — |
| $\int 1\cdot 1\,dx$ | — | — | $2\pi$ |

**正交函数系定义：** 若 $\displaystyle\int_a^b \varphi(x)\psi(x)\,dx = 0$，则 $\varphi,\psi$ 在 $[a,b]$ 上正交。

---

#### 正交性有什么用？—— 消元神器 / 声学镊子

设 $f(x) = a_0/2 + a_1\cos x + b_1\sin x + a_2\cos 2x + \cdots$

**求 $a_1$：** 两边同乘 $\cos x$，在 $[-\pi,\pi]$ 上积分：

$$\int_{-\pi}^{\pi} f(x)\cos x\,dx = \int_{-\pi}^{\pi}\Bigl(\frac{a_0}{2}\cos x + a_1\cos^2 x + b_1\sin x\cos x + \cdots\Bigr)dx$$

由正交性，右边**只剩** $a_1\cos^2 x$ 一项（其余全为 0）：

$$\int_{-\pi}^{\pi} f(x)\cos x\,dx = a_1 \cdot \pi \quad\Rightarrow\quad a_1 = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos x\,dx$$

> **物理直觉：** 复杂波形 $f(x)$ 里混了很多频率；乘以 $\cos(5x)$ 再积分，只有 $\cos(5x)$ 成分会「共振」留下，其余频率被**滤掉**——这就是傅里叶系数公式的来源。

---

| **场景二** | `sin(2x) .* sin(2x)` | 自己乘自己 → 提取系数 | $\displaystyle\int_{-\pi}^{\pi}\sin^2 2x\,dx = \pi$ |

### 1.2 傅里叶级数定义与系数公式（定理 15.2）

设 $f$ 以 $2\pi$ 为周期，在 $[-\pi,\,\pi]$ 上可积。

**傅里叶级数（教材公式 12）：**

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty}\bigl(a_n\cos nx + b_n\sin nx\bigr)$$

**符号说明：**

- **$\sim$（相似于）：** 右边是 $f$ 的傅里叶级数，**不一定处处等于** $f(x)$
- 何时能写 **$=$**？见 §1.5 收敛定理

**欧拉-傅里叶公式（官方计算公式，必背）：**

$$a_0 = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\,dx$$

$$a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos nx\,dx, \quad n = 1,2,\ldots$$

$$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\sin nx\,dx, \quad n = 1,2,\ldots$$

> 求傅里叶级数 = 把 $f(x)$ 代入上面三个积分，算系数，再代入公式 (12)。

---

#### 定理 15.2：系数公式怎么来的？（消消乐三步）

若 $f(x) = \dfrac{a_0}{2} + \sum(a_n\cos nx + b_n\sin nx)$ 且级数一致收敛，则系数必为上式。

**以 $a_k$ 为例（$b_k$、$a_0$ 同理）：**

| 步骤 | 操作 | 效果 |
|------|------|------|
| **① 锁定目标** | 两边同乘 $\cos kx$ | 准备提取 $a_k$ |
| **② 全盘扫描** | 在 $[-\pi,\,\pi]$ 上积分 | 逐项积分 |
| **③ 正交消元** | 利用公式 (7)(8) | 仅 $a_k\cos^2 kx$ 存活，其余全为 0 |

$$\int_{-\pi}^{\pi} f(x)\cos kx\,dx = a_k \int_{-\pi}^{\pi}\cos^2 kx\,dx = a_k \cdot \pi \quad\Rightarrow\quad a_k = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos kx\,dx$$

**求 $a_0$：** 直接积分（不乘），$\displaystyle\int f(x)\,dx = \frac{a_0}{2}\cdot 2\pi = a_0\pi$。

**求 $b_k$：** 同乘 $\sin kx$ 再积分，$\sin kx$ 与 $\cos nx$ 正交，只剩 $b_k\sin^2 kx$。

---

#### 隐藏彩蛋：为什么常数项是 $\dfrac{a_0}{2}$ 而不是 $a_0$？

| $n$ | 基函数 | $\displaystyle\int_{-\pi}^{\pi} (\text{基函数})^2\,dx$ |
|-----|--------|------------------------------------------------------|
| $n \ge 1$ | $\cos nx,\,\sin nx$ | $\pi$ |
| $n = 0$ | $1$（即 $\cos 0x$） | $2\pi$ ← **多一倍** |

若级数写成 $a_0 + \sum(\cdots)$，则 $a_0$ 的公式要除以 $2\pi$，和 $a_n$ 不统一。

写成 $\dfrac{a_0}{2} + \sum(\cdots)$ 后，**所有 $a_n$（含 $n=0$）** 统一为：

$$a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos nx\,dx \quad (n=0,1,2,\ldots)$$

> 记忆：不是 $a_0$ 错了，是级数首项故意乘 $\dfrac{1}{2}$，让公式好看、好背。

---

**【⚠️ 避坑】** 若 $f$ 以 $2\pi$ 为周期，积分区间任取一个完整周期（常选 $[-\pi,\pi]$ 或 $[0,2\pi]$）。

### 1.3 周期 $2l$ 的傅里叶展开

设 $f$ 以 $2l$ 为周期，令 $\omega = \dfrac{\pi}{l}$，则

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty}\left(a_n\cos\frac{n\pi x}{l} + b_n\sin\frac{n\pi x}{l}\right)$$

**系数公式：**

$$a_0 = \frac{1}{l}\int_{-l}^{l} f(x)\,dx, \qquad a_n = \frac{1}{l}\int_{-l}^{l} f(x)\cos\frac{n\pi x}{l}\,dx, \qquad b_n = \frac{1}{l}\int_{-l}^{l} f(x)\sin\frac{n\pi x}{l}\,dx$$

> **换元记忆：** $2\pi$ 情形就是 $l=\pi$ 的特例；或令 $t = \dfrac{\pi x}{l}$ 把 $2l$ 周期化为 $2\pi$ 周期。

### 1.4 奇偶性与简化（白嫖系数）

| 函数类型 | 哪些系数为 0 | 级数变成 |
|----------|-------------|----------|
| **偶函数** $f(-x)=f(x)$ | $b_n = 0$ | **余弦级数**（只有 $a_0,\,a_n$） |
| **奇函数** $f(-x)=-f(x)$ | $a_n = 0$（含 $a_0$） | **正弦级数**（只有 $b_n$） |

**为什么能「白嫖」？** 对称区间 $[-a,a]$ 上，**奇函数积分为 0**（正负面积抵消）。

**奇偶 × 三角（符号乘法）：**

| $\sin(nx)$ | 奇 $-$ | $\cos(nx)$ | 偶 $+$ |
|------------|--------|------------|--------|
| 奇 × 偶 = **奇** → 积分 $0$ | | 奇 × 奇 = **偶** → 可化 $2\int_0^{\pi}$ | |
| 偶 × 奇 = **奇** → 积分 $0$ | | 偶 × 偶 = **偶** → 可化 $2\int_0^{\pi}$ | |

| 场景 | 瞬间为 0 | 要手算（可化简） |
|------|----------|-----------------|
| $f$ **奇**，区间 $[-\pi,\pi]$ | $a_0=0,\,a_n=0$ | $b_n=\dfrac{2}{\pi}\int_0^{\pi} f(x)\sin(nx)\,dx$ |
| $f$ **偶**，区间 $[-\pi,\pi]$ | $b_n=0$ | $a_0,\,a_n=\dfrac{2}{\pi}\int_0^{\pi} f(x)\cos(nx)\,dx$ |
| **非奇非偶** 或 **$(0,2\pi)$** | 无 | $a_0,\,a_n,\,b_n$ **全算** |

**【⚠️ 避坑】捷径触发条件（两条缺一不可）：**

1. 积分区间**严格对称**：$[-\pi,\pi]$ ✓；$(0,2\pi)$ ✗（长 $2\pi$ 但不对称）
2. $f$ 在整区间上**整体**奇或偶；分段无整体奇偶（如 $f=x$ 在 $(0,\pi)$、$0$ 在 $(-\pi,0)$）→ **不能白嫖**

**考场满分话术（抄这句）：**

> 因积分区间 $[-\pi,\pi]$ 关于原点对称，且 $f(x)$ 为奇（偶）函数，由奇偶函数定积分的性质，知 $a_0=0,\,a_n=0$（或 $b_n=0$）。

### 1.5 收敛定理（定理 15.3）—— 选择题高频

> 教材手写批注：**选择题**考点。关键条件：**按段光滑**。

**定理 15.3：** 设 $f$ 以 $2\pi$ 为周期，在 $[-\pi,\,\pi]$ 上**按段光滑**，则对任意 $x$，$f$ 的傅里叶级数收敛于 $f$ 在 $x$ 处的**左右极限的算术平均**：

$$\boxed{\frac{f(x+0) + f(x-0)}{2} = \frac{a_0}{2} + \sum_{n=1}^{\infty}\bigl(a_n\cos nx + b_n\sin nx\bigr)}$$

**符号：** $f(x+0)$ = 右极限 $f(x^+)$；$f(x-0)$ = 左极限 $f(x^-)$。

**按段光滑：** 在 $[-\pi,\,\pi]$ 上可分成有限段，每段上 $f$ 可导且导数连续（比「分段连续」更强）。

---

#### 按段光滑大白话（几何翻译）

> 教材用 $f(x\pm t)$ 等极限语言写得很严，考试记住**图像**即可。

**想象：** $f$ 像几根「弯铁丝」拼成的图形——不完美，但**行为良好**。

| 允许的「瑕疵」 | 例子 | 数学说法 |
|---------------|------|----------|
| **折点** | $y=|x|$ 的尖角 | 左右导数**存在**但不相等 |
| **台阶（跳跃）** | 方波、阶跃函数 | **第一类间断点**，上下端点明确 |

**傅里叶级数「怕」的（不算按段光滑）：**

| 类型 | 例子 | 为什么不行 |
|------|------|-----------|
| 垂直渐近线 | $\dfrac{1}{x}$ 在 $x=0$ | 面积无穷，积分不存在 |
| 无穷振荡 | $\sin\dfrac{1}{x}$ 在 $x=0$ | 在一个点附近抖无穷多次 |

**教材三条性质（保证能算 $a_0,a_n,b_n$）：**

1. **可积：** 图形与 x 轴围成**有限面积** → 定积分存在  
2. **左右导数存在：** 折点处斜率不会变成「竖直悬崖」（无穷斜率）  
3. **导数也可积：** 斜率本身也围成有限面积  

> 方波：有跳跃，但每段是常数 → **按段光滑** ✓ → 定理 15.3 适用。

---

#### 收敛结论速查（考试直接写）

| 点 $x$ 的类型 | 级数和等于 |
|--------------|-----------|
| $f$ 在 $x$ **连续** | $f(x+0)=f(x-0)=f(x)$ → 和为 **$f(x)$** |
| $f$ 在 $x$ **第一类间断** | **$\dfrac{f(x+0)+f(x-0)}{2}$**（上下极限平均） |

**$\sim$ vs $=$：**

| 写法 | 什么时候用 |
|------|-----------|
| $f(x) \sim \sum(\cdots)$ | 只写傅里叶级数，不断言收敛 |
| 上式用 $=$ | 已验证**按段光滑**，写收敛结论 |

> **间断点别写 $f(x)$！** 方波跳变处级数和是中间值，不是函数取值（Gibbs 现象：间断点附近有过冲）。

## 二、通用方法

### 2.1 流水线四步法（普适解题 SOP）

傅里叶题**概念不难、计算量大**——按流水线做，不易漏步。

| 步骤 | 名称 | 做什么 | 得分点 |
|------|------|--------|--------|
| **①** | **审题偷懒** | 看区间是 $[-\pi,\pi]$ 还是 $[0,2\pi]$；判奇偶 | 奇 → 只算 $b_n$；偶 → 只算 $a_0,a_n$；**非奇非偶 → 三个都要算** |
| **②** | **硬核算系数** | 代入欧拉-傅里叶公式，分段函数**按段拆开积分** | 最耗时；$f$ 在半区间为 0 就只积另一半 |
| **③** | **组装级数** | 代回 $f(x)\sim \dfrac{a_0}{2}+\sum(a_n\cos nx+b_n\sin nx)$ | 写完通常 **≈70% 分** |
| **④** | **收敛定理收尾** | 取特殊 $x$（$0,\,\pi$，或间断点），令级数 $=$ 函数值/平均，**求数项级数和** | 求 $1+\dfrac{1}{3^2}+\dfrac{1}{5^2}+\cdots$ 这类 |

**④ 收尾三步：**

1. 选一个「好算」的 $x$ 代入第 ③ 步的级数（$\sin n\pi=0$、$\cos n\pi=(-1)^n$ 会大量消项）
2. 用定理 15.3 确定左边：连续点用 $f(x)$；间断点用 $\dfrac{f(x-0)+f(x+0)}{2}$
3. 解方程得数项级数和

**区间与奇偶对照：**

| 区间 | 奇偶性 | 要算的系数 |
|------|--------|-----------|
| $[-\pi,\pi]$ 对称 | 奇函数 | 仅 $b_n$ |
| $[-\pi,\pi]$ 对称 | 偶函数 | 仅 $a_0,\,a_n$ |
| $[0,2\pi]$ 或 非奇非偶 | — | **$a_0,\,a_n,\,b_n$ 全算** |

### 2.2 考场数学工具箱

#### 工具一：三角函数「整数倍」值（代入上下限时直接用）

| 公式 | 值 |
|------|-----|
| $\sin(n\pi) = 0$ | $n$ 为整数 |
| $\cos(n\pi) = (-1)^n$ | **最常用**，化简必备 |
| $\sin\dfrac{n\pi}{2}$、$\cos\dfrac{n\pi}{2}$ | 按 $n$ 奇偶**分类讨论**（$0,\,\pm1$） |

---

#### 工具二：分部积分法（多项式 × 三角）

**适用：** $x\cos(nx)$、$x\sin(nx)$、$x^2\cos(nx)$ 等（教材例 1、例 2 核心）

$$\int u\,dv = uv - \int v\,du$$

**口诀「反对幂指三」：** 谁当 $u$？**反**三角 > **对**数 > **幂**函数 > **指**数 > **三**角

- $x\cos(nx)$：$u=x$（求导变 1），$dv=\cos(nx)\,dx$
- $x^2\cos(nx)$：**分部积分做两次**，幂次降到 0

**常用模板（背结果更快）：**

$$\int_0^{\pi} x\sin(nx)\,dx = \frac{(-1)^{n+1}\pi}{n}, \qquad \int_0^{\pi} x\cos(nx)\,dx = \frac{(-1)^n-1}{n^2}$$

（再除以 $\pi$ 即得 $b_n,\,a_n$）

---

#### 工具三：收敛定理「断点平均」

间断点 $x_0$ 处，级数和 $S(x_0) = \dfrac{f(x_0-0)+f(x_0+0)}{2}$

> 别管端点是空心还是实心，**只看左右悬崖高度取平均**。

### 2.3 选择题模板：收敛定理考什么（秒杀两步）

**典型问法：** 给定分段函数 $f(x)$，问傅里叶级数在 $x=x_0$ 处收敛于什么？**不用算积分！**

| 步骤 | 操作 |
|------|------|
| **① 看 $x_0$ 连不连续** | 连续 → 答案 $= f(x_0)$ |
| **② 间断（或周期端点）** | 算 $f(x_0-0)$、$f(x_0+0)$，答案 $= \dfrac{f(x_0-0)+f(x_0+0)}{2}$ |

**秒杀例（方波）：** $f(x)=\begin{cases}-1,&-\pi<x<0\\1,&0\le x\le\pi\end{cases}$，问 $x=0$ 处级数和？

- 左极限 $-1$，右极限 $+1$ → 级数和 $= \dfrac{-1+1}{2} = 0$（**不是** $f(0)=1$！）

**题型 A：** $f$ 在 $x_0$ 连续 → 级数和 $= f(x_0)$

**题型 B：** $f$ 在 $x_0$ 有跳跃间断 → 级数和 $= \dfrac{L_{\text{左}}+L_{\text{右}}}{2}$

**题型 C：** 下列哪个条件保证收敛？→ 选 **按段光滑**

**题型 D：** $S(x)$ 是傅里叶和函数，$x_0$ 是间断点 → $S(x_0) \ne f(x_0)$

**方法 B：** 换元 $t = \dfrac{\pi x}{l}$，化为以 $2\pi$ 为周期的 $\tilde{f}(t)$，用 $2\pi$ 公式算完再换回 $x$。

### 2.6 经典例题：方波（奇函数三步走）

**题目：** $f$ 以 $2\pi$ 为周期，在一个周期内

$$f(x) = \begin{cases} -1, & -\pi < x < 0 \\[4pt] 1, & 0 \le x \le \pi \end{cases}$$

求傅里叶级数。

**第一步：判奇偶（省一半活）**

$f(-x) = -f(x)$ → **奇函数** → $a_0 = 0$，$a_n = 0$，**只算 $b_n$**。

---

**第二步：三步提取系数**

| 系数 | 结论 | 理由 |
|------|------|------|
| $a_0$ | $0$ | 奇函数在对称区间积分为 0 |
| $a_n$ | $0$ | 奇 × 偶 = 奇 → 积分为 0 |
| $b_n$ | **唯一要手算** | 奇 × 奇 = 偶 → 可化简为 $2$ 倍半区间积分 |

$$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\sin nx\,dx = \frac{2}{\pi}\int_0^{\pi} 1 \cdot \sin nx\,dx = \frac{2}{n\pi}\bigl(1-\cos n\pi\bigr)$$

利用 $\cos n\pi = (-1)^n$：

| $n$ | $\cos n\pi$ | $b_n$ |
|-----|-------------|-------|
| 偶数 $2,4,6\ldots$ | $1$ | $0$ |
| 奇数 $1,3,5\ldots$ | $-1$ | $\dfrac{4}{n\pi}$ |

---

**第三步：组装级数**

$$f(x) \sim \frac{4}{\pi}\left(\sin x + \frac{1}{3}\sin 3x + \frac{1}{5}\sin 5x + \cdots\right) = \frac{4}{\pi}\sum_{k=1}^{\infty}\frac{\sin\bigl((2k-1)x\bigr)}{2k-1}$$

只有**奇次**正弦项——典型方波频谱。

---

**第四步：用定理 15.3 验收敛（闭环）**

| 检验点 | 类型 | 级数和 | 验证 |
|--------|------|--------|------|
| $x = \dfrac{\pi}{2}$ | 连续点 | $f(\pi/2)=1$ | $\dfrac{4}{\pi}\left(1-\dfrac{1}{3}+\dfrac{1}{5}-\cdots\right)=1$ |
| $x = 0$ | 跳跃间断 | $\dfrac{-1+1}{2}=0$ | $\sin 0 + \sin 0 + \cdots = 0$ ✓ |

**彩蛋（Leibniz 级数）：** 由上式得

$$1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots = \frac{\pi}{4}$$

---

**物理直觉：** 方波 = 无穷多个不同频率正弦波的叠加。8-bit 游戏 BGM（如超级马里奥）就是用 $\sin x,\,\sin 3x,\,\sin 5x\ldots$ 合成「方波音色」——**数字信号（0/1）模拟成模拟信号（正弦）**。

---

### 2.7 教材例 1：锯齿波（$x$ 在 $[0,\pi]$，非奇非偶）

**题目（教材 p.64 例 1）：**

$$f(x) = \begin{cases} x, & 0 \le x \le \pi \\ 0, & -\pi < x < 0 \end{cases} \quad (\text{周期 } 2\pi)$$

$f$ **按段光滑** → 可展成傅里叶级数。注意：在 $[-\pi,\pi]$ 上**既非奇也非偶** → **$a_0,a_n,b_n$ 都要算**。

**系数（只在 $(0,\pi)$ 上非零，积分缩半区间）：**

$$a_0 = \frac{1}{\pi}\int_0^{\pi} x\,dx = \frac{\pi}{2}$$

$$a_n = \frac{1}{\pi}\int_0^{\pi} x\cos(nx)\,dx = \frac{(-1)^n-1}{n^2\pi} = \begin{cases} -\dfrac{2}{n^2\pi}, & n\text{ 奇} \\ 0, & n\text{ 偶} \end{cases}$$

$$b_n = \frac{1}{\pi}\int_0^{\pi} x\sin(nx)\,dx = \frac{(-1)^{n+1}}{n}$$

**级数（在 $(-\pi,\pi)$ 内）：**

$$f(x) = \frac{\pi}{4} - \frac{2}{\pi}\cos x + \sin x - \frac{1}{2}\sin 2x - \frac{2}{9\pi}\cos 3x + \frac{1}{3}\sin 3x - \cdots$$

**收敛（图 15-4）：** 在 $x=\pm\pi$ 处跳跃，级数和 $= \dfrac{f(\pi-0)+f(-\pi+0)}{2} = \dfrac{\pi+0}{2} = \dfrac{\pi}{2}$（**不是** $f(\pi)=\pi$）。

### 2.8 教材例 2：$x^2$ 分段 + 求 $\dfrac{\pi^2}{8}$（周期 $2\pi$，区间 $[0,2\pi]$）

**题目（教材 p.65 例 2）：**

$$f(x) = \begin{cases} x^2, & 0 < x < \pi \\ 0, & x = \pi \\ -x^2, & \pi < x \le 2\pi \end{cases}$$

**特点：** 积分区间是 $[0,2\pi]$（**非对称**）→ 三个系数全算；含 $x^2$ → **分部积分两次**。

**系数结果（直接记）：**

$$a_0 = -2\pi^2, \qquad a_n = \frac{4}{n^2}\bigl[(-1)^n-1\bigr] \;\Rightarrow\; \begin{cases} -\dfrac{8}{n^2}, & n\text{ 奇} \\ 0, & n\text{ 偶} \end{cases}$$

$b_n$ 公式较长（教材 p.66），展开后含 $\sin x,\,\sin 2x,\,\sin 3x\ldots$ 各项。

**④ 步收尾 — 在 $x=\pi$ 求数项级数和：**

- 左边（断点平均）：$\dfrac{f(\pi-0)+f(\pi+0)}{2} = \dfrac{\pi^2+(-\pi^2)}{2} = 0$
- 右边代入：$\sin(n\pi)=0$，只剩余弦项 $\cos(n\pi)=(-1)^n$

$$0 = -\pi^2 + 8\left(\frac{1}{1^2}+\frac{1}{3^2}+\frac{1}{5^2}+\cdots\right)$$

$$\boxed{\frac{\pi^2}{8} = \frac{1}{1^2}+\frac{1}{3^2}+\frac{1}{5^2}+\cdots} \quad \text{（教材公式 14）}$$

**在 $x=0$ 处**（教材公式 15）：$\dfrac{f(0-0)+f(0+0)}{2} = \dfrac{-4\pi^2+0}{2} = -2\pi^2$，代入级数同样可验证上式。

### 2.9 教材例 3：方波（振幅 $\dfrac{\pi}{4}$，Gibbs 现象）

**题目（教材 p.67 例 3，电子工程方波）：**

$$f(x) = \begin{cases} -\dfrac{\pi}{4}, & -\pi \le x < 0 \\[6pt] 0, & x = 0 \\[6pt] \dfrac{\pi}{4}, & 0 < x < \pi \end{cases}$$

**奇函数** → $a_0=a_n=0$，只算 $b_n$：

$$b_n = \frac{2}{\pi}\int_0^{\pi}\frac{\pi}{4}\sin(nx)\,dx = \frac{1}{2n}\bigl[1-(-1)^n\bigr] = \begin{cases} \dfrac{1}{n}, & n\text{ 奇} \\ 0, & n\text{ 偶} \end{cases}$$

**级数（$x \ne k\pi$）：**

$$f(x) = \sin x + \frac{1}{3}\sin 3x + \frac{1}{5}\sin(5x) + \cdots$$

与 §2.6 的 $\pm1$ 方波**同型**，只是振幅从 $\dfrac{4}{\pi}$ 缩成 $1$（函数值 $\dfrac{\pi}{4}$ 而非 $1$）。

**图 15-7（Gibbs 现象）：** $n=1$ 只有 $\sin x$；$n=2$ 加 $\dfrac{\sin 3x}{3}$；$n=7$ 已接近方波，但**间断点附近仍有过冲**——这是正常现象，不是算错。

## 三、易错点

### ⚠️ 避坑 1：首项写成 $a_0$ 而不是 $\dfrac{a_0}{2}$

级数形式是 $\dfrac{a_0}{2} + \sum(\cdots)$，$a_0$ 本身已经乘了 $\dfrac{1}{\pi}$ 的积分，别再多除一次。

---

### ⚠️ 避坑 2：奇函数还去算 $a_0$

奇函数在 $[-\pi,\,\pi]$ 上 $f(x)=-f(-x)$，$a_0 = \dfrac{1}{\pi}\int f(x)\,dx = 0$，$a_n$ 也全为 0。

---

### ⚠️ 避坑 3：间断点处期望级数等于 $f(x)$

例如方波在间断点，级数和是 **上下极限的平均**，不是函数值本身（Gibbs 现象附近会有过冲）。

---

### ⚠️ 避坑 4：$2\pi$ 与 $2l$ 公式混用

周期是 $2l$ 时，$\cos$ 里必须是 $\dfrac{n\pi x}{l}$，系数分母是 $l$，不是 $\pi$。

---

### ⚠️ 避坑 5：积分区间不是完整一个周期

系数公式里的积分必须在一个**完整周期**上算；若 $f$ 以 $2\pi$ 为周期，不能只在 $[0,\,\pi/2]$ 上积分（除非利用对称性严格说明）。

---

### 题 1(1) $f(x)=x$：同一函数，两种区间 → 两种级数

**核心对比：** $(-\pi,\pi)$ 能白嫖奇函数；$(0,2\pi)$ **不能**，三项全算。

#### (i) $-\pi < x < \pi$（对称区间 + 奇函数）

| 系数 | 结果 | 理由 |
|------|------|------|
| $a_0$ | $0$ | 奇函数在对称区间 |
| $a_n$ | $0$ | 奇 × 偶 = 奇 |
| $b_n$ | $\dfrac{2}{n}(-1)^{n+1}$ | 奇 × 奇 = 偶，分部积分 |

**分部积分（你的草稿同款）：**

$$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} x\sin(nx)\,dx = -\frac{1}{n\pi}\bigl[x\cos(nx)\bigr]_{-\pi}^{\pi} + \frac{1}{n\pi}\int_{-\pi}^{\pi}\cos(nx)\,dx = \frac{2}{n}(-1)^{n+1}$$

**级数：**

$$f(x) = x = 2\sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n}\sin(nx), \quad x\in(-\pi,\pi)$$

展开：$2\bigl(\sin x - \dfrac{\sin 2x}{2} + \dfrac{\sin 3x}{3} - \cdots\bigr)$

**④步彩蛋：** 取 $x=\dfrac{\pi}{2}$，$f(\pi/2)=\dfrac{\pi}{2}$，$\sin\dfrac{n\pi}{2}$ 只在 $n=4k\pm1$ 非零 →

$$\frac{\pi}{4} = 1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots \quad \text{（Leibniz 级数）}$$

---

#### (ii) $0 < x < 2\pi$（非对称 → 全算）

| 系数 | 结果 |
|------|------|
| $a_0$ | $2\pi$（$\dfrac{1}{\pi}\int_0^{2\pi}x\,dx$） |
| $a_n$ | $0$（分部积分 + $\sin n\pi=0$） |
| $b_n$ | $-\dfrac{2}{n}$ |

**级数：**

$$f(x) = x = \pi - 2\sum_{n=1}^{\infty}\frac{\sin(nx)}{n}, \quad x\in(0,2\pi)$$

> **对比记忆：** 同是 $f(x)=x$，$(-\pi,\pi)$ 无常数项；$(0,2\pi)$ 多出 $\pi$ 常数项，且 $\sin$ 系数符号全为负。区间不同 → 系数不同 → 级数不同。

---

### 题 2 积分区间平移不变（证明题模板）

**结论：** $f$ 以 $2\pi$ 为周期、可积，则对任意实数 $c$，

$$\frac{1}{\pi}\int_c^{c+2\pi} f(x)\cos(nx)\,dx = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos(nx)\,dx$$

（$\sin$ 项、$a_0$ 同理。）

**证法（求导法，最简）：** 令 $F(c)=\displaystyle\int_c^{c+2\pi} f(x)\cos(nx)\,dx$。

$$F'(c) = f(c+2\pi)\cos\bigl(n(c+2\pi)\bigr) - f(c)\cos(nc) = f(c)\cos(nc+2n\pi)-f(c)\cos(nc) = 0$$

故 $F(c)$ 与 $c$ 无关。取 $c=-\pi$ 即得。

> **大白话：** 周期函数在一个完整周期上「扫」出的面积，从哪开始扫都一样。

### 题 4 $f(x+\pi)=-f(x)$ → 只有奇次谐波

**条件含义：** $f$ 仍以 $2\pi$ 为周期，但在半周期处**反号**（如方波）。

**在 $(-\pi,\pi)$ 上傅里叶级数的性质：**

| 系数 | 结论 |
|------|------|
| $a_0$ | $=0$ |
| $a_n,\,b_n$ | $n$ **偶数** 时 $=0$；$n$ **奇数** 时可能非零 |

**级数只含奇次项：** $\cos x,\,\sin x,\,\cos 3x,\,\sin 3x,\,\ldots$（无 $\cos 2x,\,\sin 2x$ 等）

**证法要点（代换 $u=x+\pi$）：**

$$\int_{-\pi}^{0} f(x)\cos(nx)\,dx = -\int_{0}^{\pi} f(u)(-1)^n\cos(nu)\,du$$

与 $\int_0^{\pi}$ 部分相加，系数因子为 $1-(-1)^n$：$n$ 偶 → $0$；$n$ 奇 → 非零。

### 题 7(1) $f(x)=\dfrac{\pi-x}{2}$，$0<x<2\pi$

**区间 $(0,2\pi)$** → 不判奇偶，三项全算（但这题 $a_0,\,a_n$ 恰好全为 0）。

| 系数 | 结果 | 计算要点 |
|------|------|----------|
| $a_0$ | $0$ | $\int_0^{2\pi}(\pi-x)\,dx = 0$ |
| $a_n$ | $0$ | $\int_0^{2\pi}(\pi-x)\cos(nx)\,dx = 0$（分部积分） |
| $b_n$ | $\dfrac{1}{n}$ | $\dfrac{1}{2\pi}\int_0^{2\pi}(\pi-x)\sin(nx)\,dx$ |

**级数：**

$$f(x) = \frac{\pi-x}{2} = \sum_{n=1}^{\infty}\frac{\sin(nx)}{n}, \quad x\in(0,2\pi)$$

即 $\dfrac{\pi-x}{2} = \sin x + \dfrac{\sin 2x}{2} + \dfrac{\sin 3x}{3} + \cdots$

**验收：** $x=\pi$ 时左边 $=0$，右边 $\sin n\pi=0$ ✓。
